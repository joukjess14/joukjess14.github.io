🎨 中国古代桥梁文化博物馆 —— 数字化交互展示系统
2026年计算机设计大赛参赛作品 探索千年营建智慧，重塑指尖上的古桥文化

🏛️ 项目愿景
中国古桥不仅是交通设施，更是融合了建筑力学、工程哲学与美学艺术的文化瑰宝。本项目致力于通过Web数字孪生、动态数据可视化与AI智能导览技术，打破时空限制，为大众构建一个可交互、沉浸式的“纸上桥梁博物馆”，让尘封在典籍中的造桥智慧焕发新生。

✨ 核心亮点
1. 🌐 3D数字化名桥展厅 (exhibition.html)
数字孪生渲染：基于 Google Model-Viewer 引擎，实现赵州桥、洛阳桥等名桥的高精度 3D 模型交互展示。

多维交互：支持 360° 旋转、缩放，并内置 AR（增强现实）接口，用户可一键将古桥“搬”入现实空间。

2. 🗺️ 动态地理寻踪系统 (index.html)
智能动态地图：深度定制 ECharts 地图组件，动态展示全国各省份古桥遗存分布。

地域风貌关联：点击不同省份可实时调取该地区特色桥梁档案（如江南廊桥、西南索桥），展现“因地制宜”的营建思想。

3. 🤖 AI 智能导览员“桥小博”
智慧对话：集成大语言模型（LLM）API，化身首席导览员。

流式交互：支持实时流式输出，能够精准回答关于桥梁结构（如“种蛎固基”）、历史变迁及科学家事迹的专业问题。

4. 📚 文献档案馆与先贤名录 (books.html, scientists.html)
典籍数字化：收录《营造法式》、《水经注》等核心文献，以卡片化形式解析古籍中的建筑规范。

匠心传承：多维展示李春、蔡襄、郭守敬等先贤的事迹，通过“勾股定理应用”等细节还原古代工程科学。

🛠️ 技术栈
前端框架：Tailwind CSS (响应式布局与原子化样式管理)

可视化引擎：ECharts 5.0 (地理信息与动态统计图表)

3D 技术：Google Model-Viewer & WebGL

交互动画：Cubic-bezier 贝塞尔曲线过渡、Intersection Observer 滚动视差

数据驱动：JSON 结构化存储桥梁与科学家档案

📂 文件结构
Plaintext
├── index.html          # 博物馆主殿（门户中心）

确保环境支持 Live Server（推荐使用 VS Code 插件）。
